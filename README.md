### Описание данных
<u> Таблица products: </u>
- product_id - id товара
- level1 - категория
- level2 - подкатегория
- name - наименование товара

<u> Таблица orders: </u>
- order_id - номер чека
- accepted_at - дата и время чека
- product_id - id товара
- quantity - кол-во товара в чеке
- regular_price - регулярная цена
- price - текущая цена
- cost_price - закупочная цена
